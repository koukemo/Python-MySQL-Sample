# Python-MySQL-Sample

Sample of writing to mysql in docker with python.

## Installing

---

- Docker
- Docker Compose
- Python 3.9.x
    - mysql-connector-python (pip)

## Tests

---

Run the following command:

Terminal1
```shell
cd Python-MySQL-Sample/db
docker-compose up -d
```

Terminal2
```shell
cd Python-MySQL-Sample
python3 main.py
```
